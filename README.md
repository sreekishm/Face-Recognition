# Face-Recognition
A real-time face recognition system built with Python that identifies individuals from a webcam feed by comparing against a database of known faces.

# Features

- Encodes and stores known faces from a local image directory
- Real-time face detection and recognition via webcam
- Displays the recognized person's name on the live video feed
- Lightweight and fast — uses image downscaling for performance

# Tech stack
- Python
- OpenCV — video capture and image processing
- face_recognition — face detection and encoding (built on dlib)
- pickle — serializing face encodings
- Numpy compatibility <2.0
